# Soketi on PipeOps

This example deploys self-hosted version of [Soketi](https://soketi.app). 

## ✨ Features

- Soketi
- Pusher API

## 💁‍♀️ How to use

- Click the Pipeops button 👆
- Enjoy hyper-scalable, extremly fast websockets.
